# Multi-Algo Cryptotool

Данный проект выполнен в рамках расчетно-графической работы по дисциплине «Программирование», НГТУ, АВТФ АБс-524 Взоров Д.К.
Multi-Algo Cryptotool - это консольная утилита для шифрования и дешифрования файлов с использованием трех алгоритмов: Квадрат Полибия (16x16), блочная перестановка и RSA.

## Для работы с проектом используются команды:

### Клонирование репозитория

Команда: git clone https://github.com/EternalSeele/Multi-Algo-Cryptotool.git

### Сборка проекта

Выполните полную сборку проекта: компиляция объектных файлов, создание динамических библиотек и сборка исполняемого файла

Команды:

make clean

make all

### Тестирование

Перед использованием программы рекомендуется запустить тесты, чтобы убедиться в корректной работе всех алгоритмов.

Команда: ./test_crypto

### Справка по использованию программы

Справка содержит описание всех поддерживаемых алгоритмов, режимов работы и доступных опций.

Команда: ./multi_algo_cryptotool --help

## Примеры использования

### Генерация ключа RSA
./multi_algo_cryptotool -a rsa -m generate-key -s rsa_key.bin

### Шифрование файла (Polybius)
./multi_algo_cryptotool -a polybius -m encrypt -i input.txt -o input_enc.enc

### Дешифрование файла (RSA)
./multi_algo_cryptotool -a rsa -m decrypt -k rsa_key.bin -i input_enc.enc -o output_dec.txt
